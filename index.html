My location not working?

iPhone: Settings → Privacy & Security → Location Services. Turn it On, then scroll to Safari Websites and set it to While Using the App.

Android: Settings → Location → turn On. Then in Chrome: ⋮ menu → Settings → Site settings → Location → Allow.

Then close the browser fully and reopen this app.
